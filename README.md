# Todo CLI

A simple command-line todo application built with Node.js and Commander.

## Features

- Add todos with task descriptions
- Delete todos by index
- Mark todos as done

## Installation

```bash
npm install
```

## Usage

```bash
# Add a todo
node index.js add "Your task here"

# Delete a todo
node index.js delete <index>

# Mark todo as done
node index.js done <index>
```
